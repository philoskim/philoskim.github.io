영어사전 검색 이용 안내

<검색시 유의사항>

  • 검색하고자 하는 영어 단어 또는 숙어를 검색창에 입력한 후, [Enter] 키를 누르거나, 검색을 원하는 영어사전 사이트의 버튼을 누른다.
  • [Enter] 키를 누르면, 기본적으로 영어사전 사이트의 검색 결과가 화면에 출력된다.
  • 원하는 버튼을 누르면, 해당 영어사전 사이트의 검색 결과가 화면에 출력된다.
  • 검색어 입력시, 기존에 입력한 단어가 입력창에 지워지지 않고 남아 있으면, Esc 키나 아래쪽 화살표 키를 누르면 지워진다.
  • 새로운 검색어를 입력하기 위해 검색창을 클릭하면, 직전에 입력했던 검색어가 사라진다. 사라진 직전 검색어를 다시 이용하고 싶으면, 위쪽 화살표 키를 누른다.
  • 영어사전 검색 사이트 버튼에는, 회색 버튼과 노란색 버튼의 구별이 있다. 회색 버튼의 경우에는, 검색 결과가 바로 검색창 아래의 화면에 출력되지만, 노란색 버튼의 경우에는, 검색 결과가 새로운 페이지에 출력된다는 것을 표시하기 위해 버튼의 배경색을 다르게 하였다. 검색 결과가 새로운 페이지에 표시되는 경우(노란색 버튼을 눌렀을 때), 검색어를 다시 입력하기 위해 브라우저의 backward 버튼을 눌러줘야 하는 불편함이 뒤따른다. 회색 버튼의 경우가 검색하기에 더 편리함에도 불구하고, 프로그래밍상의 기술적인 제약으로 인해 어쩔 수 없이 새로운 페이지에 출력하도록 하였다.
  • 유용한 영어사전 사이트가 발견될 때마다, 이곳에 그 사이트들을 계속 추가해 나갈 것이므로, 이 사이트의 소스를 복사해 사용하는 것 보다는, 브라우저의 '즐겨찾기' 기능이나, '북마크(bookmark)' 기능을 이용해 접속하는 것이 더 나을 것이다.

<각각의 사전에 대한 간략한 소개>

  • : '네이버' 검색 사이트에서 제공하는 영어사전이다.
  • : '다음' 검색 사이트에서 제공하는 영어사전이다.
  • : 'YBM Sisa' 사이트에서 제공하는 영어사전이다.
  • : bluedic 사이트에서 제공하는 영어사전이다.
  • : 한국인 개인이 운영하는 영어 학습사전이다. 숙어 정리가 잘 되어 있고, 제공되는 예문이 충실한 편이다.

  • : 글자 그대로 idiom(관용구)의 정확한 의미를 찾고자 할 때 유용하다.
  • : 관용구의 어원을 찾고자 할 때 이용하면 좋다. 영어 단어의 경우에는 어원이 많이 연구되어 있는 편이나, 관용구의 경우에는 그 어원이 잘 정리되어 있는 사이트를 찾기가 힘들다. 그래서 구글 검색을 통해 관용구의 어원을 찾을 수 있도록 하였다.
  • : 글자 그대로 collocation(연어)을 찾아 준다. 예를 들어, '강의하다'를 영어로 표현하고자 할 때, 'lecture' 앞에 어떤 동사를 써야 하는지 막막해지는 경우가 있다. 이때 이 사이트에서 'lecture'라는 검색어를 입력하면, give a lecture 등과 같이 lecture라는 단어와 함께 빈번하게 연결되어 쓰이는 표현들의 목록을 확인할 수 있다. 다시 말하자면, 특정 단어를 입력할 때 그 단어와 자주 연결되어 사용되는 단어들의 목록을 확인할 수 있다. 참고로, 이 사이트를 검색할 때에는 숙어가 아닌 '단어'를 검색어로 입력해야 한다.
  • : 글자 그대로 slang 표현이 잘 정리되어 있는 사이트이다.
  • : 1500년대에서 2000년대 최근까지 해당 단어 또는 숙어의 사용 빈도를 그래프로 출력해준다.
  • : 정규식(regular expresions)을 이용해 단어를 검색해 준다. 예를 들면, -ly 로 끝나는 모든 형용사를 찾고 싶을 때, 이 사전을 이용하면 유용하다.
  • : 입력한 영어 단어들의 발음을 들려준다. 특히, 인명(人名)의 발음을 확인하고 싶을 때 유용하다.

  • : 때로는 구글 검색의 결과가 영어사전 검색의 결과보다 더 좋은 경우가 많아서, 구글 검색을 추가하였다.
  • : 이 사이트는 본인이 가장 즐겨 찾아보는 영어사전 사이트이다. 단어나 숙어 및 관용구의 설명이 아주 간단명료하면서도 핵심을 찌르는 방식으로 서술되어 있고, 등장하는 예문도 의미를 파악하기에 아주 적합한 것들로 구성되어 있다.
  • : Merriam-Webster 영엉 사전 사이트이다.
  • : 이 사이트의 장점은, 각 단어가 사용되는 예를 `구체적인 상황'을 들어 설명하고 있어, 어떤 상황에서 단어가 사용되는지에 대한 구체적인 정보를 제공하고 있다는 점이다. 아울러 단어의 어원도 잘 설명되어 있는 편이고, 단어의 의미를 기술하기 위해 사용되는 단어들이 대체로 평이하다는 것도 장점이다. 핵심적인 단어의 의미 위주로 간략하게 기술되어 있다는 것도 또다른 장점이다. 단어의 의미가 너무 많이 서술되어 있으면, 자세하다는 장점은 있지만, 어떤 의미가 핵심적인 의미로 사용되는지를 파악하기 어렵게 만드는 단점도 지니고 있기 때문이다.
  • : yourdictionary.com에서 제공하는 영엉 사전 사이트이다. 이 사전은 해당하는 표현이 어떤 상황에서 사용되는지 구체적인 예를 제시해 준다는 점에서, 단어나 숙어에 대한 이해를 도와준다.
  • : 영어 단어의 어원을 자세하게 설명해 주는 사이트이다.
  • : 일반인들의 자발적인 참여를 통해 인터넷 상에서 공동으로 만들어 가고 있는 open source 영어 사전 사이트이다.
  • : 이 사이트는 단어의 의미를 서술할 때, 아주 평이한 단어들을 이용해 설명되어 있어, 영어를 외국어로 배우는 사람들에게 적합하다. 단어의 의미가 아주 많이 서술되어 있어, 위에 언급한 'vocabulary' 사전을 보완해 주는 성격도 지닌다. 숙어, 관용구, 동의어, 반의어, slang, 어원등 다방면에 걸쳐 많은 정보가 담겨 있는 아주 유용한 사이트이다.
  • : 위의 'dictionary' 사이트와 유사한 성격의 사이트이다.
  • : 글자 그대로 동의어/반의어가 잘 정리되어 있는 곳이다. 특정 단어나 숙어에 대한 영어로 된 설명을 읽어도 잘 이해가 안되는 경우가 종종 있는데, 이때 동의어나 반의어를 찾아 보면 의외로 쉽게 이해되는 경우가 많다.
  • : Collins Cobuild 영어 사전의 online 판이다.
  • : 단어 또는 숙어 하나를 입력하면, 수십 개의 사전 목록이 화면에 출력된다. 위에 소개한 영어 사전 사이트들도, 주로 이곳에서 검색을 통해 알게 된 사이트들을 접근하기 쉽도록 따로 뽑아 놓은 것이다. 특히 이곳은 일반 영어사전들 뿐만 아니라 법률, 경제, 예술, 과학, 기술, 컴퓨터, 스포츠 등과 같은 전문 용어 사전들도 함께 검색해 준다.
